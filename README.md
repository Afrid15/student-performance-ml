# Mini Data Science Project: Predicting Student Exam Performance

## Objective
This project analyzes student-related factors and predicts final exam grade using machine learning.

## Dataset
UCI Student Performance Dataset

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

## Steps
1. Data loading
2. Data cleaning
3. Exploratory data analysis
4. Linear regression modeling
5. Results and interpretation

## Main Findings
- Previous Academic Performance is the Strongest Predictor
- Study Time Positively Influences Academic Success
- Past Failures Negatively Affect Final Performance
- Absenteeism Has a Moderate Negative Impact
- Dimensionality Reduction Using PCA
- Machine Learning Model Comparison

## Files
- `Data_Analysis.ipynb` → analysis notebook
- `Data_Analysis.pdf` → code blocks
- `Report.pdf` → final report
